# 🌱 Empowering Minds, Shaping Innovators

Technology has changed my life, and I wanted to give youth(future innovators) the same opportunity to explore it.

After returning from the TechGirls Exchange Program, I launched Empowering Minds, Shaping Innovators, a project to introduce everyone to the world of STEM. Many of them had never knew what STEM actaully is.

This project is more than giving them a know how to STEM but actually helping them with their future, I mean the fields, the careers and the pathways for which they are confused. It is about creating a space where youth learners feel curious, ask questions, make decision, and discover what is good for them and what should they pursue.


Technologies Used:
- HTML5
- CSS3
- JavaScript
- Bootstrap
- EarSketch

How to Run the Project?
1. Clone this repository.
2. Open the project folder.
3. Launch `stemProject.html` in your browser.

That's it!
